---

## 🚀 Deploy Instructions
1️⃣ Open [Remix IDE](https://remix.ethereum.org/)  
2️⃣ Upload `contracts/JOMA.sol`  
3️⃣ Compile using **Solidity 0.8.x**  
4️⃣ Deploy using MetaMask (Injected Web3)  
5️⃣ Choose BNB Chain (Mainnet) and confirm gas fee  

✅ After deploy, add your contract address to this README for transparency.
